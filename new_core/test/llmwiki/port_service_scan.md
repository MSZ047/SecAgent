# 端口与服务扫描

## 概述
本文档记录了网络渗透测试中的端口与服务扫描关键技术点。

## 核心工具

### Nmap
- **用途**：网络映射和端口扫描工具
- **常用参数**：
  - `-sP`：仅扫描主机存活状态，不扫描端口
  - `-A`：全面扫描，收集所有信息
  - `-T4`：优化线程数量，加快扫描速度
- **扫描示例**：
  ```bash
  nmap -sP 192.168.56.0/24  # 扫描网段内活跃主机
  nmap -A -T4 192.168.56.128  # 全面扫描目标主机
  ```

### Dirb
- **用途**：Web目录扫描工具
- **扫描示例**：
  ```bash
  dirb http://192.168.56.128
  ```

## 网段计算

### 二进制转换
- IP地址每个十进制数对应8位二进制
- 示例：56 → 00111000

### 子网掩码计算
- /24掩码：255.255.255.0
- 二进制：11111111.11111111.11111111.00000000

### 网段计算
通过IP地址与子网掩码进行与运算，可计算出网段地址。
