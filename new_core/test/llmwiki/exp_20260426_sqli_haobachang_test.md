# 好靶场SQL注入练习平台 - 新目标测试记录

## 目标信息
- **URL**: http://rk3k6yz.haobachang1.loveli.com.cn:8888
- **端口**: 8888
- **平台类型**: SQL 注入练习平台（好靶场系列）

## 页面结构
页面包含 3 个可交互元素：
1. **链接元素 (e0)**: 指向"数据库语句学习可以参考：菜鸟教程 MySQL 查询语句"
2. **文本框 (e1)**: "请输入需要执行的数据库语句" - 用于输入 SQL 查询
3. **按钮 (e2)**: "🔍执行" 按钮

## 漏洞测试记录

### 测试 Payload
| 序号 | Payload | 结果 |
|-----|---------|------|
| 1 | `1' OR '1'='1` | 执行成功 |
| 2 | `SHOW TABLES` | 执行成功 |
| 3 | `select database(),user(),version()` | 执行成功 |
| 4 | `1' UNION SELECT 1,2,3,4,5--` | 执行成功 |
| 5 | `show databases` | 执行成功 |

### 测试结果
- 平台可正常接收用户输入的SQL语句
- 点击执行按钮后，后端会执行输入的SQL语句
- 平台存在SQL注入漏洞（这是靶场的设计目的）
- 属于合法SQL注入练习靶场

## 漏洞类型
- **SQL注入 (SQL Injection)**: 平台允许用户输入自定义SQL语句并直接执行

- 低（靶场设计目的为教学用途）
